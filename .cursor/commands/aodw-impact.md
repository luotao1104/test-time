# 更新影响分析

更新 RT 的 impact.md 文档，分析改动的影响范围。

## 功能
- 分析模块级影响
- 分析行为影响
- 分析数据模型影响
- 分析 API 影响
- 评估风险和回退计划
- 更新 impact.md 文档

## 影响分析维度
1. **模块级影响**：哪些模块的行为变化，跨模块依赖
2. **行为影响**：功能变化、副作用、回归风险、数据流变化
3. **数据模型影响**：schema 更改、数据迁移要求、兼容性
4. **API 影响**：签名更改、向后兼容性、客户端影响
5. **运行时影响**：性能、并发、安全影响

- Spec-Full RT 的必需步骤
