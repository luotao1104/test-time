# RT 一致性检查

检查 RT 文档的一致性和完整性，确保所有文档匹配且符合规范。

## 功能
- 验证 meta.yaml 正确性
- 检查 index.yaml 同步状态
- 验证模块 README 匹配实现
- 检查不变量是否被违反
- 验证 RT 文档完整性
- 检查 tests.md 是否包含验证步骤
- 验证治理要求

## 检查项
- [ ] meta.yaml 已更新
- [ ] RT/index.yaml 已更新
- [ ] spec/spec-lite 已完成
- [ ] plan/plan-lite 已完成
- [ ] invariants.md 已验证
- [ ] impact.md 已验证
- [ ] tests.md 已完成
- [ ] 模块 README 已更新
- [ ] 治理规则已满足

## 使用场景
- RT 完成前的一致性验证
- 定期检查 RT 文档状态
- 修复文档不一致问题
