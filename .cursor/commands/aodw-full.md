# Spec-Full 模式

将当前 RT 切换到 Spec-Full 模式，适用于复杂、跨模块的改动。

## 功能
- 切换到完整文档模式
- 创建完整的 spec.md、plan.md、impact.md、invariants.md、tests.md
- 深度影响分析
- 先设计后编码
- 全面测试定义

## 使用场景
- 跨模块调整
- 数据模型更改
- API 契约演进
- 架构决策
- 性能/并发/安全影响
- 大规模重构

## 必需文档
- meta.yaml
- intake.md
- decision.md
- spec.md
- plan.md
- impact.md
- invariants.md
- tests.md
- changelog.md
