# 生成测试计划

为 RT 生成或更新测试计划文档。

## 功能
- 生成测试用例矩阵
- 定义输入用例和预期输出
- 标注回归测试点
- 添加异常/错误/边界场景测试
- 提供性能/安全验证策略（如需要）
- 更新 tests.md 文档

## Spec-Lite 测试
- 必需手动验证步骤
- tests.md 包括：最小步骤、预期结果、失败情况

## Spec-Full 测试
- 必需自动化测试建议
- tests.md 包括：测试矩阵、集成测试计划、回归测试列表、性能/安全检查

## 使用场景
- RT 创建时生成初始测试计划
- 实现过程中更新测试用例
- RT 完成前验证测试完整性
